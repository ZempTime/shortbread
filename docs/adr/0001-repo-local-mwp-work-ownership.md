# Repo-local MWP owns work state and evidence

Shortbread defines its Model Workspace Protocol in `docs/agents/mwp.md`: GitHub issues and pull requests coordinate across sessions, initiative workspaces contain the work, and each workspace `RUN.md` is authoritative for live authority and stage state. `agents/` owns the reusable factory, and every completed run records a harvest decision, because branches and tracker labels cannot prove that reasoning, verification, or learning occurred.
